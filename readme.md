 Sobre o Projeto

Portfólio pessoal desenvolvido com **HTML, CSS e JavaScript puro**, apresentando minha trajetória como **Backend Software Engineer**. O design prioriza uma experiência fluida com animações suaves e layout responsivo para todos os dispositivos.


Funcionalidades

-  Animação de digitação no hero
-  Barra de progresso de leitura
-  Reveal com blur e zoom ao rolar a página
-  Cards de habilidades com entrada em cascata
-  Botão de copiar e-mail com feedback animado
-  Navbar responsiva com menu mobile
-  Fundo animado com gradiente dinâmico
- Download direto do currículo em PDF

 Tecnologias

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura e semântica |
| CSS3 | Animações, variáveis, Grid e Flexbox |
| JavaScript | IntersectionObserver, animações e interações |
